# Sovereign AI Redaction Demo

A dynamic, instantly branded, fully client-side PII redaction sandbox built with Next.js 16.

## What This Shows
- Automatic company branding from any domain (logo + color)
- 100% client-side air-gapped redaction (nothing leaves the browser)
- Realistic System Audit Terminal with persistent streaming logs
- Regex-based PII masking (SSN, names, etc.)
- Clean 4-panel enterprise dashboard

Built as a portfolio piece for Solutions Architect / Forward Deployed Engineer roles.

## Tech Stack
- Next.js 16 (App Router), React 19, TypeScript
- Tailwind CSS
- Deployed on Vercel

## Run Locally
`ash
npm install
npm run dev
`
Open http://localhost:3000. Enter any company domain (e.g. arva.ai), click Generate Demo, then click PROCESS SECURELY.
